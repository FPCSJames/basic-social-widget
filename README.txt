== Description ==

Displays a row of clickable social media icons. Best in footer sections.

== Installation ==

Install from Plugins > Add New > Upload Plugin, or:

1. Upload the zip file containing this plugin to your plugins directory, and unpack it
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Activate the FPCS Basic Social widget.
